# Ulotki (flyer PDF templates)

Flyer PDFs go here, one per variant combination, named `<key>.pdf` to match the
keys registered in `src/lib/flyers/flyer-template-registry.ts`.

Status: **not yet delivered.** The generation architecture is scaffolded
(`src/lib/flyers/*`) with TODOs; drop the PDFs here and register them when ready.

Each flyer will be filled with: school name, address, insurance period, selected
variant(s)/options, agent/contact data, and premium (if available), then saved
alongside the policy documents.
